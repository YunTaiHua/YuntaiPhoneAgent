# ==================== 安装指南 ====================
Phone Agent 依赖安装指南

## 1. 基础安装（必须）
pip install -r requirements.txt

## 2. TTS功能安装（需要语音合成）
pip install -r tts_requirements.txt

## 3. 完整安装（包含所有功能）
pip install -r requirements.txt -r tts_requirements.txt

## 4. Windows系统额外安装
pip install -r windows_requirements.txt

## 5. 开发环境安装
pip install -r dev_requirements.txt

## 6. 可选功能安装
pip install -r optional_requirements.txt

# ==================== 系统要求 ====================

## 操作系统
- Windows 10/11 64位（推荐）
- Linux Ubuntu 20.04+（支持）
- macOS 12+（部分功能可能受限）

## Python版本
- Python 3.8 - 3.11（推荐3.9）

## 硬件要求
- CPU: Intel i5 或同等AMD处理器（推荐i7）
- 内存: 8GB（推荐16GB）
- 存储: 至少10GB可用空间
- GPU: NVIDIA GTX 1060或更高（TTS加速）

## 必要软件
1. ADB工具（Android调试桥）
2. Scrcpy（手机投屏工具）
3. Git（版本控制）
4. FFmpeg（音频处理）

# ==================== 安装步骤 ====================

## Windows安装步骤
1. 安装Python 3.9
2. 安装Git
3. 安装FFmpeg并添加到PATH
4. 下载Scrcpy并配置路径
5. 运行：pip install -r requirements.txt

## Linux安装步骤
1. sudo apt update
2. sudo apt install python3.9 python3-pip
3. sudo apt install git ffmpeg adb
4. pip3 install -r requirements.txt

## 验证安装
python -c "import customtkinter; import zhipuai; import torch; print('✅ 核心依赖安装成功')"

# ==================== 常见问题 ====================

## Q: TTS模块导入失败
A: 确保安装了所有tts_requirements.txt中的包

## Q: ADB连接失败
A: 检查手机USB调试是否开启

## Q: 音频播放无声音
A: 检查pyaudio和系统音频驱动

## Q: GUI界面显示异常
A: 确保customtkinter版本正确
