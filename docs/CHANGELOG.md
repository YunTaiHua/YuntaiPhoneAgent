## v1.0.x 系列

### 1.0.9 (2025.12.21)
- 添加重试机制，若聊天记录提取失败则尝试重新提取

### 1.0.8 (2025.12.21)
- 增加可查看和清空日志的功能

### 1.0.7 (2025.12.20)
- 修改 phone_agent 提示词，要求准确返回头像位置信息

### 1.0.6 (2025.12.18)
- 修改消息判断规则：左侧有头像->对方消息，右侧有头像->我方消息

### 1.0.5 (2025.12.15)
- 优化 phone_agent 提示词，增加消息判断准则，根据气泡颜色判断，并加入 conversation_history.json 文件构建上下文和记录日志

### 1.0.4 (2025.12.15)
- 优化双 AI 协作:
  - GLM-4 智能判断任务类型（然后执行以下五种任务）
  - 自由聊天，仅 GLM-4 响应
  - 手机操作，phone_agent 响应
  - 单次回复：phone_agent 提取聊天记录，GLM-4 生成回复内容，phone_agent 将回复内容发出
  - 持续回复：phone_agent 提取聊天记录，GLM-4 生成回复内容，phone_agent 将回复内容发出，phone_agent 提取聊天记录，GLM-4 判断是否有新消息，是则生成回复内容让 phone_agent 发出，否则让 phone_agent 继续提取聊天记录
  - 复杂操作：GLM-4 直接将原指令传达给 phone_agent

### 1.0.3 (2025.12.15)
- 改变字体颜色，GLM-4 输出为金色，phone_agent 输出为绿色，最终输出为蓝色

### 1.0.2 (2025.12.14)
- 引入 GLM-4 构建了双AI系统，同时实现了自由聊天和手机操作，GLM-4 做决策，phone_agent 执行

### 1.0.1 (2025.12.14)

## 版本命名规则

- **主版本号 (Major)**: 重大架构变更或不兼容的 API 修改
- **次版本号 (Minor)**: 新增功能或较大改进
- **修订号 (Patch)**: Bug 修复或小改进
